# Homepage Visual Story Plan

## Emotional Arc Through Color & Design

This document maps how visual design supports the narrative arc from top to bottom.

---

## Section 1: HERO
**Narrative Beat:** The Promise - "Build your way with Amplifier"

**Emotional Tone:** 
- Freedom
- Expansion
- Possibility
- Beauty
- Openness

**Visual Treatment:**
```
Background: Light, airy, spacious
- Base: canvas (light warm white)
- EmergenceField particle system (active, flowing)
- Gradient: subtle from-canvas/40 via-transparent to-canvas/60

Text Colors:
- Headline: text-ink (strong, clear)
- Body: text-ink (readable, confident)
- Code block: Highlighted, inviting

Visual Cues:
- Generous whitespace
- Particles move freely, organically
- Open, breathable layout
```

## Chladni Resonance - The Central Storytelling Element

**Key Principle:** The Chladni particle system is THE central storytelling element. Like a real vibrating plate, each section creates a distinctive resonance pattern that reinforces the emotional and conceptual beat. The particles flow smoothly between patterns, creating a subtle visual narrative that runs throughout the entire experience.

### Pattern Meanings

| Section | Resonance Pattern | Emotional Quality | What It Communicates |
|---------|-------------------|-------------------|----------------------|
| **Hero** | Simple cross (n=1, m=2) | Open, welcoming, clear | "This is accessible. Come in." |
| **Problem** | Scattered chaos (n=1, m=1, high chaos) | Confusion, no structure | "This is messy. No clarity." |
| **Answer** | Three pillars (n=3, m=3) | Liberation, clarity emerging | "Here's the structure. This makes sense." |
| **Platform** | Four quadrants (n=4, m=4) | Complete, organized | "It's comprehensive. All parts work together." |
| **Demo** | Diagonal flow (n=3, m=5) | Action, motion | "This is working. Live." |
| **Bundle** | Perfect grid (n=5, m=5) | Elegant unity | "One file. Everything unified." |
| **Ecosystem** | Dense network (n=6, m=7) | Abundance, connectivity | "Whole network. You're not alone." |
| **CTA** | Central convergence (n=6, m=6) | Resolution, focus | "Here. This is the action." |

### Critical Transitions

**HERO ‚Üí PROBLEM:** Structure dissolves into chaos (gradual decay)
**PROBLEM ‚Üí ANSWER:** Chaos snaps into three clear pillars (dramatic, liberating - THE KEY MOMENT)
**ANSWER ‚Üí PLATFORM:** Three pillars expand into four quadrants (organic growth)
**BUNDLE ‚Üí ECOSYSTEM:** Simple unity becomes dense network (evolution)

### Visual Subtlety
- Opacity: Never above 0.5 - patterns support, don't dominate
- Always in motion, never static
- Transitions: 2 seconds, cubic easing (like real frequency changes)
- Particle size: 2-3px - visible but not distracting
