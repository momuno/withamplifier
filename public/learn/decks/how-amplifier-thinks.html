Autonomy

When does it wait for you?

Acts autonomously when:
  • It has a clear next step
  • Tools are available
  • The action is safe
  • It hasn't hit the turn limit
Waits for you when:
  • It needs clarification or a decision
  • It's completed what it can do
  • A tool requires your approval
  • It reaches the turn limit
Debugging

What's normal? What's not?

Normal behavior:
  • Reading files, running commands
  • Multiple tool calls per turn
  • Asking clarifying questions
  • Trying alternate approaches when stuck
  • Delegating to specialized agents
Something's wrong if:
  • It submits prompts you didn't write
  • Raw internal objects appear
  • Actions don't relate to your request
  • It loops endlessly on the same error
  • Tools fail with cryptic messages
The Point

You can see it thinking

Every decision is logged. Every tool call is visible.
